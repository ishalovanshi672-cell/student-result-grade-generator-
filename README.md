# Student Result and Grade Generator

This is a simple web-based project that calculates student result, percentage and grade automatically.

## Features
- Enter student details and subject marks
- Automatic calculation of total marks
- Percentage calculation
- Grade generation
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- PHP
- MySQL

## How It Works
1. User enters student name, roll number and marks
2. System calculates total and percentage
3. Grade is generated based on percentage
4. Result is displayed on the screen

## Grading System
- 90% - 100% : A+
- 75% - 89% : A
- 60% - 74% : B
- 45% - 59% : C
- Below 45% : Fail

## Conclusion
This project helps in reducing manual work and provides fast and accurate student result generation.
